# 🤝 Contributing to CKAD Kubernetes Practice Lab

Thank you for your interest in contributing! This project welcomes contributions from the community.

## ⭐ How You Can Help

### 1. Star This Repo
If this project helped you prepare for CKAD, please **star ⭐ the repo** — it helps others find it!

### 2. Report Issues
Found a bug in a YAML manifest or a better way to write it?
→ Open an [Issue](https://github.com/NAVEED261/ckad-kubernetes-project/issues)

### 3. Suggest New Examples
Want to add a new CKAD topic example? Open a PR!

**Good topics to contribute:**
- Init containers
- Persistent Volumes (PV/PVC)
- Resource Quotas / LimitRange
- HorizontalPodAutoscaler (HPA)
- ServiceAccount + RBAC
- Secrets management

### 4. Improve Documentation
Better comments, more examples, clearer README — all PRs welcome!

## 📋 Contribution Guidelines

1. Fork the repo
2. Create a branch: `git checkout -b feature/add-hpa-example`
3. Add your YAML with **inline comments explaining every field**
4. Submit a Pull Request with a clear description

## 🌍 Who Is This For?

Anyone preparing for:
- **CKAD** (Certified Kubernetes Application Developer)
- **CKA** (Certified Kubernetes Administrator)
- **CKAD/CKA bootcamps and study groups**
